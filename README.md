# Get Hyped

A modern, responsive website built with Next.js 16 and Tailwind CSS.

## Live Demo & Repository

🔗 **Live Site:** [https://get-hyped-omega.vercel.app/](https://get-hyped-omega.vercel.app/)

📦 **GitHub Repository:** [https://github.com/joykbiswas/get-hyped](https://github.com/joykbiswas/get-hyped)
## Tech Stack

- **Framework:** Next.js 16.2.3
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## Features

- Responsive design
- Smooth animations with Framer Motion
- Modern UI components
- Fast page loads with Next.js

## Getting Started

### Installation

```bash
# Clone the repository
git clone <repository-url>

cd get-hyped

# Install dependencies
pnpm install

```
